 1. Создать директорию для проекта
 2. В данной директории необходимо создать виртуальное окружение
 python3 -m venv название
 активировать окружение
 sourse название_окружения/bin/activate
 3. В папке проекта необходима создать файл зависимостей requirements.txt и описать все библиотеки, которые будут использованы в проекте
 4. Установить все заависимости в виртуальное окружение
 pip install -r requirements.txt

 5. Создаем Django проект:
 django-admin startproject название_преокта .

 6. Создаем приложение проекта
 ./manage.py startapp название

 7. Создаем БД для проекта
 psql потом CREATE DATABASE название БД;

 8. Настройка проекта (settings.py)

 Указываем приложения в INSTALLED_APPS
 Указываем настройки БД
 Указываем настройки языка, часового пояса

 9. Создаем модель пользователя
 в settings добавляем настройку AUTH_USER_MODEL

 10. Определяем остальные модели

 11. Делаем миграции
 ./manage.py makemigrations потом
 ./manage.py migrate

 12. Создать админа
 ./manage.py createsuperuser
